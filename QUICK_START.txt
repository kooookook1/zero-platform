📋 خطوات التشغيل:

1. تحميل المشروع:
   - اضغط على الرابط أعلاه
   - سيتم تحميل ملف: zero-platform-complete.tar.gz

2. فك الضغط:
   tar -xzf zero-platform-complete.tar.gz
   cd zero-platform

3. تثبيت المتطلبات:
   pip install flask flask-cors

4. تشغيل المشروع:
   python3 simple_app.py

5. فتح المتصفح:
   http://localhost:5000
