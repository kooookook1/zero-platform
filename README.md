## 🎯 نظرة عامة

**ZERO Platform** هي منصة ويب شاملة مصممة خصيصاً للمختصين في الأمان السيبراني والاختراق الأخلاقي. تتميز المنصة بواجهة عربية RTL كاملة وتصميم عصري يدعم أكثر من 15 أداة اختراق مشهورة.

### ✨ الميزات الرئيسية

- 🌐 **واجهة عربية RTL كاملة** - تصميم مخصص للمستخدمين العرب
- 🛠️ **15+ أداة اختراق مشهورة** - جميع الأدوات تعمل فعلياً
- 🎨 **تصميم متجاوب وجذاب** - يعمل على جميع الأجهزة
- ⚡ **سهولة الاستخدام** - واجهة بديهية ومبسطة
- 🔒 **آمن ومحمي** - للأغراض التعليمية فقط

## 📦 التحميل والتثبيت

### 🚀 التحميل السريع

```bash
# تحميل أحدث إصدار
wget https://github.com/kooookook1/zero-platform/archive/refs/tags/v1.0.0.tar.gz

# فك الضغط
tar -xzf v1.0.0.tar.gz
cd zero-platform-1.0.0

# تثبيت المتطلبات
pip install flask flask-cors

# تشغيل المشروع
python3 simple_app.py
```

### 🔄 استنساخ المستودع

```bash
git clone https://github.com/kooookook1/zero-platform.git
cd zero-platform
pip install -r requirements.txt
python3 simple_app.py
```

### 🌐 الوصول للمنصة

بعد التشغيل، افتح المتصفح على:
```
http://localhost:5000
```

## 🛠️ الأدوات المتاحة

### 🔍 الصفحة الأولى: اكتشاف الثغرات في المواقع
- **SQLMap** - اختبار حقن SQL
- **Nikto** - فحص الثغرات على سيرفرات الويب
- **DirB/Gobuster** - اكتشاف الأدلة والملفات المخفية
- **WhatWeb** - معرفة التقنيات المستخدمة في الموقع
- **WPScan** - فحص مواقع ووردبريس

### 🌐 الصفحة الثانية: فحص الشبكات والخدمات
- **Nmap** - مسح البورتات والخدمات
- **Masscan** - سكان سريع جداً
- **Hping3** - فحص الاستجابات
- **DNSEnum** - فحص DNS
- **SNMP-Check** - فحص خدمات SNMP

### 🔐 الصفحة الثالثة: كلمات السر والتشفير
- **Hydra** - تجربة brute-force
- **John the Ripper** - كسر الهاشات
- **Hash-Identifier** - التعرف على أنواع الهاش
- **CeWL** - توليد قوائم كلمات مرور
- **Crunch** - توليد كلمات مرور مخصصة

### 📱 الصفحة الرابعة: أدوات أندرويد
- **MSFVenom** - توليد APK ملغم بجلسة Meterpreter
- **APKTool** - تفكيك وإعادة بناء تطبيقات APK
- **Objection** - تخطي الحمايات وتشغيل أوامر
- **Drozer** - استغلال ثغرات التطبيقات
- **JADX** - ديكومبايل لملفات Java

### 📡 الصفحة الخامسة: هجوم واي فاي
- **Aircrack-ng Suite** - كسر WPA/WPA2 عبر handshake
- **Reaver** - هجوم WPS brute-force
- **Wifite** - واجهة مؤتمتة لكل aircrack/reaver
- **HCXDumpTool** - هجوم PMKID المباشر
- **Bettercap** - هجوم MITM على الواي فاي

### 🔵 الصفحة السادسة: هجوم بلوتوث
- **Bluelog** - سكان لأجهزة بلوتوث القريبة
- **Blue_Hydra** - سكان BLE متطور
- **BTScanner** - معلومات متقدمة عن الأجهزة
- **Ubertooth** - هجمات على الطبقات المنخفضة
- **CarWhisperer** - استغلال أجهزة بلوتوث الصوتية

### 🕸️ الصفحة السابعة: التحكم بالاتصال وهجمات MITM
- **Ettercap** - هجمات MITM وتسميم ARP
- **DNSSniff Suite** - التقاط كلمات مرور وبروتوكولات
- **MITMProxy** - اعتراض وتعديل طلبات HTTP/HTTPS
- **SSLStrip** - تحويل HTTPS إلى HTTP لسرقة البيانات
- **NetSniff-ng** - حزمة أدوات متقدمة لتحليل الشبكات
